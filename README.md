Data Visualization – Virtual Internship (Infosys Springboard)

This repository contains the work completed as part of the Data Visualization Virtual Internship offered by Infosys Springboard.
The internship focuses on building a strong foundation in data visualization concepts, tools, and practical implementation through structured milestones and hands-on exercises.

Internship Overview

Platform: Infosys Springboard

Domain: Data Visualization

Mode: Virtual Internship

Objective:
To understand data visualization principles and apply them to transform raw data into meaningful insights using appropriate visualization techniques and tools.

Repository Structure

The repository is organized milestone-wise, with each folder containing the tasks, datasets, and outputs related to that specific milestone.

Key Learnings

Fundamentals of data visualization
